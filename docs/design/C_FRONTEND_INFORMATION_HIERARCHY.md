# 灵析 C 端前端信息层级方案

**状态：** 已实现并通过用户验收
**日期：** 2026-07-17

**本地验证：** Web 全量单测 **98 files / 437 tests passed**，TypeScript 类型检查、源码 ESLint、生产构建与 `git diff --check` 均通过；已在 **1440×900** 桌面端和 **390×844** 手机端逐页检查持仓、分析、市场、发现、日报，未发现页面级横向溢出，默认折叠和主操作可见性符合本方案。390px 下五页主操作实测高度为 **44–46.5px**；市场详情、完整板块榜单与分页展开控件均以稳定 `aria-controls` 关联受控区域。

## 1. 目标

在不削弱后端研究能力、风险守卫和证据可追溯性的前提下，让普通用户在每个页面先回答三个问题：

1. 现在是什么状态？
2. 我需要做什么？
3. 为什么，以及主要风险是什么？

默认视图统一遵守：**一句结论 → 三个关键数字 → 一个主操作 → 专业证据按需展开**。

## 2. 竞品调研结论

| 产品 | 值得借鉴 | 需要避免 | 对灵析的启发 |
|------|----------|----------|--------------|
| 蚂小财 | 先给个性化回答，再逐步补充解释；入口围绕用户问题而非工具名 | 对话式入口若缺少事实边界，容易让用户过度信任结论 | 首页和报告先回答“今天是否需要动”，证据与边界紧随其后 |
| 且慢 | 用“四笔钱”等生活目标翻译专业资产配置，主动过滤短期噪声 | 目标体系过重会增加首次使用成本 | 用白话标签表达风险，不在默认视图堆叠夏普、Beta、Factor IC |
| 天天基金 | 数据、交易、资讯覆盖全面，适合主动检索的熟练用户 | 工具、榜单、资讯同时抢占注意力，容易形成数据终端式拥挤 | 完整榜单保留，但必须由用户主动展开 |
| 小倍养基 | 板块与基金工具丰富，便于研究和交叉核对 | 多维排行与专业指标容易成为默认信息墙 | 市场页先给市场判断和五个相关方向，再展示全量研究工具 |

## 3. 已确认的信息架构

- 保留五个现有入口：持仓、分析、市场、发现、日报。
- 登录后默认进入持仓；不恢复已经删除的简报 Tab。
- 保留“静谧蓝海 / editorial research desk”视觉语言，暖金只用于关键结论或钱相关高光。
- 移动端页面标题必须紧凑，内容不能被大标题和重复说明挤到首屏之外。

## 4. 页面契约

### 持仓

- 默认：总资产、今日收益、基金列表、一个“新增持有”主操作。
- 移动卡：基金、今日收益、持有收益；板块只作为“相关方向”弱标签。
- 按需：排序、批量加减仓、校准持仓。

### 盈亏分析

- 默认：白话状态、我的收益率/同期大盘/领先落后三个数字、影响最大的基金、趋势、日历、持仓分布。
- 按需：历史最深下跌、分散程度、相关性、因子评分、Factor IC 与证据总览。
- 白话主标签必须优先，专业名仅作为次级说明：

| 专业名 | 默认标签 |
|--------|----------|
| 最大回撤 | 历史最深下跌 |
| 年化波动率 | 涨跌有多剧烈 |
| Beta | 跟大盘有多像 |
| Alpha | 有没有跑赢大盘 |
| 主力净流入 | 大资金流向 |

### 市场观察

- 默认：一句市场判断、上涨/下跌/涨停三个数字、持仓相关或领先的五个方向。
- 按需：上涨下跌比例条、停牌与涨跌停细项、九档基金分布、完整板块榜单、资金历史。

### 发现基金

- 默认输入：推荐目标、关注方向、预算、扫描按钮。
- 按需输入：流程轨道、机会/稳健策略、AI 表达附录、系统校验说明。
- 默认结果：最多三只高价值候选；每张卡先展示状态、动作、金额、一个理由、一个风险。
- 按需结果：交易条件、质量门、同类分位、基准、完整证据与历史复盘。

### 投研日报

- 生成前：当前偏好摘要与“生成今日操作建议”主按钮。
- 生成后：今天是否需要行动、基金动作、金额、核心理由、主要风险。
- 按需：风格与 Prompt、详细风险参数、要闻、轮动、调仓模拟、回测、诊断和证据。

## 5. 验收标准

- 390px 移动端首屏能看见页面任务或主操作，不出现横向页面滚动。
- 默认视图不得展示 Factor IC、Beta、Alpha、完整候选质量表或九档分布。
- 所有被折叠内容仍可访问，并保留原有数据时点、失败态和重试能力。
- 主按钮触控高度至少 44px；展开按钮带 `aria-expanded`，关联内容带稳定 `aria-controls`。
- Web 全量单测、TypeScript 类型检查、源码 ESLint、生产构建通过；浏览器覆盖持仓、分析、市场、发现、日报的桌面和 390px 移动端。
