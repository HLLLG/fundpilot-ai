# 荐基与日报金融评估 Phase 0+1

## 已确认目标

在适当性、流动性、集中度和回撤约束下，最大化费后、风险调整后的预期总收益。方向命中率保留为兼容指标，不再作为唯一成功标准。

## Phase 0：金融口径纠偏

- 基金收益统一采用“官方日增长率优先、单位净值比值兜底”的总收益指数，避免分红除息被误判为亏损。
- 组合最大回撤的权益曲线显式包含初始值 `1.0`，首日下跌不再漏记。
- 夏普使用日频超额收益均值除以样本波动率后乘 `sqrt(252)`；索提诺使用全部观测日的下行二阶矩；Alpha 使用日频 CAPM 残差均值年化。
- 20～59 个交易日仍可展示风险指标，但明确标为 `short_window`、`annualization_reliable=false`；60 日起进入标准年化窗口。
- 集中度以当前实际组合市值为分母。计划投入额只在不存在正市值持仓时作为空组合兼容值。
- 成本基准浮亏告警改为 `PORTFOLIO_COST_BASIS_LOSS` / `HOLDING_COST_BASIS_LOSS`，不再冒充历史峰谷最大回撤。

## Phase 1：影子评估控制面

### 版本与窗口

- 新决策：`decision_policy.2026-07.v5`、`decision_strategy.post_guard.v2`。
- 评估策略：`strategy_evaluation.2026-07.v1`，固定 `shadow_record_only`。
- 日报与荐基统一预登记 T+5 / T+20 / T+60；旧日报 T+1 observation 仍可由结算器处理，但新事件不再创建 T+1。
- 日报指标版本升级为 `forward_total_return_v2` / `daily_forward_total_return_v2`，与历史单位净值口径隔离。

### 成熟结果

每个成熟 observation 同时记录：

1. 总收益方向、用户假设费后正收益、完整基金合同基准毛超额、费后超额；
2. MAE（路径最不利变动）、MFE（路径最有利变动）、路径最大回撤；
3. 日收益 CVaR 95%，至少 20 个日收益观测才可用；
4. 相对“不行动”的每单位变更名义本金增量收益；没有冻结仓位变化或费用时明确不可用；
5. 荐基的同板块质量优先、低费率和确定性随机候选基线；候选、成本或后续净值不足时只降低覆盖率。

### 不行动反事实的边界

结果按“本次变更的名义本金”表达，不冒充组合收益率。日报减仓比例通常相对当前持仓，荐基加仓比例可能相对预算，两者不可直接汇总成同一种组合贡献，除非未来冻结统一的组合权重基准。

### 点时与不可变性

- 策略上下文、费用、基准、候选基线、模型/Prompt/策略版本随 DecisionEvent 冻结。
- 目标收益只使用执行日起基金自身估值日序列；缺失、未成熟和 provider 错误保持不可评价。
- 终态 OutcomeObservation 继续走现有 revision 与冲突检测，不回写历史 V2 事件。
- 新字段位于既有 JSON 契约内，不增加数据库列或 schema 迁移；历史事件按原 horizon/version 保持可读。

## 验收标准

- 分红除息样本不会产生虚假负收益或虚假回撤。
- 首日为负、零波动、短样本和缺失样本均有确定结果或明确不可用原因。
- 新日报/荐基事件只创建 T+5/20/60，旧 T+1 可继续结算。
- MAE/MFE/CVaR、不行动和候选基线均随 observation 持久化；缺输入不进入命中或失败分母。
- 用户侧只展示影子结果，不自动修改推荐、仓位、Prompt、Guard 或模型权重。
