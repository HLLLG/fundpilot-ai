# 灵析前端品牌替换设计

## 目标

将前端对用户展示的“好基灵 / FundPilot”统一替换为“灵析 / LINGXI”，并将正式站点地址统一为 `https://hllingxi.cn`。页面标题、描述、分享元数据、品牌组件、落地页文案、桌面通知和功能提示必须保持一致。

## 品牌定义

- 中文品牌名：`灵析`
- 英文辅助名：`LINGXI`
- 产品定位：`灵析 AI 基金研究台`
- 正式站点：`https://hllingxi.cn`
- SEO 标题：`灵析 | AI 基金研究台`
- SEO 描述：`灵析：智能识别基金持仓，追踪市场与板块变化，结合量化证据生成个性化投研分析与风险提示。`

## 实现边界

新增一个无副作用的前端品牌常量模块，作为品牌名、产品名、站点地址和元数据文案的唯一来源。`BrandMark`、根布局元数据及包含品牌称呼的组件改为读取该模块；不改变页面布局、配色和现有抽象图标。

继续保留 `fundpilot-*` localStorage/sessionStorage 键、API 环境变量、数据库名、CloudBase 环境 ID 和内部服务名。这些值属于兼容性或基础设施标识，重命名会使已有用户会话和缓存失效，且不影响用户看到的新品牌。

## 验收标准

1. 前端可见内容、页面元数据和桌面通知中不再出现“好基灵”或“FundPilot”。
2. 品牌组件显示“灵析”，英文辅助名显示“LINGXI”。
3. canonical 与 Open Graph URL 指向 `https://hllingxi.cn`。
4. 已有 `fundpilot-*` 浏览器存储键保持不变。
5. 前端单元测试、类型检查、Lint 和生产构建全部通过。
